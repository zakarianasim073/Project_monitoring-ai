## 2025-05-14 - Login Accessibility and Quick Access
**Learning:** Associating labels with inputs using `htmlFor` and `id` is a fundamental accessibility requirement that is often overlooked. Additionally, providing "Quick Login" buttons for demo accounts significantly reduces friction for first-time users and reviewers.
**Action:** Always ensure form labels are programmatically linked to their inputs and consider adding one-click demo access for internal/review environments.

## 2025-05-14 - Vite Infrastructure
**Learning:** A missing `index.html` in a Vite project prevents the dev server and build process from functioning, even if all components are present.
**Action:** Verify project entry points and basic configuration files (`index.html`, `vite.config.ts`) when setting up a new or broken frontend environment.
