## 2026-04-12 - Improving Login Accessibility & UX

**Learning:** Initial repository state had mismatched build tools (Next.js vs Vite) and missing basic accessibility features like linked labels and focus rings.

**Action:** When working on a project with ambiguous infrastructure, first standardize the `package.json` to match the actual source code (Vite SPA in this case) and add essential UX touches like password toggles and interactive demo credentials to improve the first-time user experience.
