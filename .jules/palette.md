# Palette Journal 🎨

## 2025-05-15 - Login Page Accessibility and Quick Access
**Learning:** High-impact micro-enhancements for `Login.tsx` include semantic `htmlFor`/`id` linking for accessibility, implementing a `showPassword` toggle with Eye/EyeOff icons, and adding role-specific 'Quick Login' buttons. These features significantly improve first-time user experience and streamline development/demo workflows.
**Action:** Always implement these surgical UX improvements on login screens. Ensure auxiliary buttons (like password toggles) use `type="button"` to avoid form submission.
